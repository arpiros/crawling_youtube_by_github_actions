# crawling_youtube_by_github_actions
## Required
### env
|name| desc|
|-|-|
|WEBHOOK_URL|discord webhook url|
|YOUTUBE_URL|search youtube url|

### pip install
```shell
pip install discord-webhook
pip install get-chrome-driver --upgrade
pip install -r selenium/requirements.txt
```

## RUN
```shell
python webhook.py
```
